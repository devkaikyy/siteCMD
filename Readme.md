# 🌐 Página Web Responsiva

## 📌 Sobre o projeto
Este projeto é uma página web responsiva desenvolvida para se adaptar a diferentes tamanhos de tela, garantindo uma boa experiência tanto em dispositivos móveis quanto em desktops.

## 🚀 Tecnologias utilizadas
- HTML
- CSS
- JavaScript

## 📱 Responsividade
- Layout adaptado para celulares
- Tablet
- Desktop
- Uso de media queries

## ⚙️ Funcionalidades
- Menu responsivo com animação
- Interações com JavaScript
- Manipulação do DOM

## 🎯 Objetivo
Praticar a criação de layouts modernos e responsivos, aplicando boas práticas de desenvolvimento front-end.
